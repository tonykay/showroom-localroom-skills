== Technical requirements

=== Software

* Claude Code (latest version) with an active API key
* A terminal (macOS Terminal, iTerm2, Linux terminal, or WSL)
* A text editor (VS Code, Vim, or any editor you prefer)
* Git (for cloning the workshop repo)

=== Optional -- Localroom

This workshop includes **Localroom**, a browser-based split-pane dev environment.
To use it, you also need:

* ttyd (`brew install ttyd` on macOS)
* code-server (`brew install code-server`)
* Podman or Docker (for building Antora content)

Run `./localroom.sh start` from the repo root to launch it.

=== No external dependencies

* No Kubernetes cluster required
* No Ansible installation required
* No cloud accounts needed
* All exercises use sample error logs provided in the workshop repo

== Environment setup

=== Pre-workshop checklist

* **Claude Code installed** -- Verify with `claude --version`
* **API key configured** -- Verify with `claude` (should start a session)
* **Workshop repo cloned** -- Clone this repository locally
* **Text editor ready** -- Any editor that can create `.md` files

=== Setup validation

Run these commands to verify your setup:

[source,role="execute"]
----
# Verify Claude Code
claude --version

# Verify you can create files in the skills directory
mkdir -p ~/.claude/commands
echo "Setup verified" && echo "Ready to build skills"
----
