= Getting Started with Skills

Welcome to the Getting Started with Skills workshop!

== What you'll learn

In this workshop, you will:

* Understand what agent skills are and why they matter as an open standard
* Compare skills with MCP servers and /slash commands to know when to use each
* Build a suite of practical SRE diagnostic skills from scratch
* Capture tribal knowledge as reusable, shareable skills
* Distribute skills across teams and agentic frameworks

== Who this is for

This workshop is designed for developers and engineers who are already using Claude Code and want to extend it with custom skills.

**Experience level**: Intermediate -- you should be comfortable with Claude Code basics.

== Prerequisites

Before starting this workshop, you should have:

* Claude Code installed and configured with an active API key
* A terminal environment (macOS, Linux, or WSL)
* Basic familiarity with Claude Code interactions
* A text editor

== Workshop environment

You'll work entirely on your local machine using Claude Code and a text editor.
This workshop includes **Localroom** -- run `./localroom.sh start` for a browser-based split-pane environment with a terminal and code editor alongside these instructions.

No cloud clusters or external services are required.

== Estimated time

This workshop will take approximately **90 minutes** to complete.

You can work at your own pace.
