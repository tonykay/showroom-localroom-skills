= Demo Module 1: Your First Pipeline
:source-highlighter: rouge
:toc: macro
:toclevels: 1

In this module you will show how a Tekton pipeline automates the full build-test-deploy
cycle for ACME Corp's application.

== Part 1: Creating the Pipeline

=== Know

ACME currently deploys manually — a developer packages the application, uploads to a server,
and another team member runs deployment scripts. This takes 3-5 days and fails 20% of the time.

*What you will show*: A pipeline that does in 5 minutes what takes ACME 3 days manually.

*Business value*: 60x faster deployments, zero manual steps, consistent every time.

=== Show

. Log in to the cluster and switch to the ACME project:
+
[source,role="execute"]
----
oc login --server={openshift_api_url} --token={openshift_token}
oc project acme-pipelines
----

. Apply the pre-built pipeline definition:
+
[source,role="execute"]
----
oc apply -f https://raw.githubusercontent.com/rhpds/acme-pipelines/main/pipeline.yaml
----

. Confirm the pipeline was created:
+
[source,role="execute"]
----
oc get pipeline acme-build-deploy
----

Expected result: Pipeline appears with status "Not started".

[NOTE]
====
*Presenter Tip:* Point out that this single `oc apply` replaces the 3-5 days of manual packaging and deployment ACME's team currently does — and it's repeatable every time.
====

== Part 2: Running the Pipeline

=== Know

The pipeline connects to ACME's Git repository and automatically builds on every commit.
This replaces 3-5 days of manual work.

=== Show

. Start the pipeline run:
+
[source,role="execute"]
----
tkn pipeline start acme-build-deploy \
  --param git-url=https://github.com/acme/app.git \
  --param image-url=image-registry.openshift-image-registry.svc:5000/acme-pipelines/app:latest \
  --workspace name=source,claimName=acme-source-pvc \
  --showlog
----

. In a second terminal, watch the pods spin up in real time:
+
[source,role="execute"]
----
watch oc get pods -l tekton.dev/pipeline=acme-build-deploy
----

. When the run completes, confirm the new image is deployed:
+
[source,role="execute"]
----
oc get pipelinerun --sort-by=.metadata.creationTimestamp | tail -1
oc get deployment acme-app -o jsonpath='{.spec.template.spec.containers[0].image}'
----

Expected result: Application updated, pipeline run shows "Succeeded".

[NOTE]
====
*Presenter Tip:* While the pipeline runs, narrate each task — git-clone, buildah build, deploy — and tie each back to a manual step ACME was doing before. "This task alone saved their team two days."
====
